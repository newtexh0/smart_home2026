System Architecture

This prototype represents a secure, accessible, and user-centered interface for modern Cyber-Physical Systems.

Technical Stack:
  • Frontend: HTML5 (Semantic), CSS3 (Responsive)
  • Interaction: Vanilla JavaScript (No Frameworks)
  • State Management: LocalStorage API (Client-side DB)
  • Accessibility: WCAG 2.1 Level AA Compliant

HCI Research Principles Applied

Visibility of Status

Users are constantly informed about system state (e.g., Loading Spinners, Toggles).

Match Real World

Interfaces mimic physical objects (3D Switches, Thermostat) for intuitive control.

Inclusive Design

Tools for Color Blindness, Dyslexia, and Motor impairments are integrated.
